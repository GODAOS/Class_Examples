1. 변수
	- 데이터(값)를 저장할 때 사용되는 메모리 기억 공간
	- 데이터를 담는 상자 → 크기 ▶ Data Type(자료형)
	- 기억 공간에는 이름이 있어야 한다 ▶ Identifier(식별자)
	※ 변수 이름 (식별자) 작성 규칙
		- 반드시 영문자 소문자 시작 (영문자, '_', '$'만 사용)
		- 예약어(KeyWord)는 사용 불가
		- 띄어쓰기는 사용할 수 없다.
		- 가급적이면 이름에 의미가 나타나도록 작성 하자

	※ 선언, 할당, 초기화, 나열 , 재할당
	
2. Data Type (자료형) : 변수의 크기와 저장되는 데이터의 종류를 결정
	① 기본자료형(Primitive Data Type)
		- 정수형 : byte(1), short(2), int(4), long(8)
		- 실수형 : float(4), double(8)
		- 문자형(단일문자) : chat(2)
		- 부울형(논리형) : boolean(1)
		
	② 사용자 정의 자료형 : 참조형 자료(Reference Type) → 4Byte
		- 클래스 타입
		- 인터페이스 타입
		- 배열(열거) 타입
		
※ 윈도우 10 단축키
	- Windows 로고 키 + R : 시작 → 실행
	- Windows 로고 키 + E : 내 PC
	- Windows 로고 키 + D : 바탕화면 바로가기 (모두 최소화)
	- Windows 로고 키 + ← : 창을 분할하여 왼쪽 정렬
	- Windows 로고 키 + → : 창을 분할하여 오른쪽 정렬
	- Alt + Tab : 작업 전환창을 이용하여 프로그램 전환
	- Alt + Esc : 프로그램 전환
